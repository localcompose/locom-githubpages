# Install, Setup, Test & Cleanup

=== "Linux"
    ```bash
    # Example Linux install
    curl -sSL https://example.com/locom/install.sh | bash
    ```

=== "macOS"
    ```bash
    brew install locom
    ```

=== "Windows"
    ```powershell
    winget install locom
    ```

---

## Testing the tool
```bash
locom --version
```

## Cleanup
```bash
locom uninstall
```
